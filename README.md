# Data-Science-For-Health-System---Project

Questo progetto effettua un'analisi comparativa tra le cause di morte **neonatali** (0–27 giorni) e le morti **infantili** (0–11 mesi di vita).  

Il confronto viene eseguito a livello globale su un arco temporale di oltre 20 anni (2000–2021), con l’obiettivo di evidenziare differenze statistiche, temporali e geografiche tra i due gruppi.

---

## 🎯 Obiettivo

1. Analisi esplorativa dei dataset utilizzati  
2. Visualizzazione delle statistiche principali  
3. Analisi temporale e spaziale comparata dei due gruppi  
4. Studio della normalità dei dati mediante test di **Kolmogorov-Smirnov**  
5. Analisi delle distribuzioni nei vari Paesi tramite test non parametrici (**Kruskal-Wallis**)  
6. Tecniche di clusterizzazione mediante **Elbow Method** e **Principal Component Analysis (PCA)**
